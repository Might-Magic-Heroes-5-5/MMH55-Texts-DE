Melodias Gefühle können so flatterhaft sein wie die eines Kobolds. Tatsächlich verliebt sie sich so schnell, dass sie Dutzende von Männern mit gebrochenem Herzen zurückgelassen hat, die sich nach ihrer Aufmerksamkeit sehnen. Aber sie will ihnen nicht wehtun. Sie würde sie weiterhin lieben, wenn sie könnte, aber sie wurde einfach nicht so geboren.
