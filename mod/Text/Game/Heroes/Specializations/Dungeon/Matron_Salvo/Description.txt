Ordensherrin: Schattenhexen, Schattenmatriarchen und Schattengebieterinnen haben die Chance, einen freien zusätzlichen Schuss auf das ausgewählte Ziel des Helden zu machen. Die Chance liegt bei 30%+Heldenstufe*1% und kann auf mehrere Stapel angewendet werden. Der Schuss verwendet nur eine Munition und kann bei Schadens- und Fluchzauber ausgelöst werden, selbst wenn ein Feind benachbart ist. Er wird auf die Einheit mit dem Cursor ausgelöst, auf die ein Massenzauber mit Flächenwirkung verwendet wird. Wird der Zauber durch einen Zauberspiegel umgelenkt, schießen die Hexen auf das umgelenkte Ziel (auch auf sie selbst).
